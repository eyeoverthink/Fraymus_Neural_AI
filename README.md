# Fraymus Neural & AI

Neural networks and AI applications.

## Applications

- **BicameralMind** - Bicameral Mind - Dual Hemisphere Consciousness
- **HyperSynapse** - HyperSynapse - Ternary Logic Crystal with Wormholes
- **FraymusCoder** - Fraymus Coder - Ollama Puppeteer with Self-Correction
- **TachyonRouter** - Tachyon Router - Retro-Causal Predictive Engine
- **SovereignMind** - Sovereign Mind - The Ghost in the Shell
- **NeuroQuantDemo** - NeuroQuant Demo - 10,000D Hyperdimensional Computing + NCA

## Dependencies

- LWJGL 3.3.3 (OpenGL, GLFW, STB)
- ImGui Java 1.86.11
- JOML 1.10.5
- Neural cores
- LLM bridges (Ollama, OpenAI)

## Build

```bash
cd Asset-Manager
./gradlew build
```

## Run Neural & AI Apps

```bash
./gradlew runBicameral       # Run Bicameral Mind
./gradlew runHyperSynapse    # Run HyperSynapse
./gradlew runCoder           # Run Fraymus Coder
./gradlew runTachyon         # Run Tachyon Router
./gradlew runSovereign       # Run Sovereign Mind
./gradlew runNeuroQuant      # Run NeuroQuant Demo
```

## Requirements

- Java 21
- Gradle 9.2+
- Ollama (for LLM integration)
